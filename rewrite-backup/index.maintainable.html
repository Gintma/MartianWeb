about

Mono is brutal, readable, and built for interface work.

[ 01 ] system

Martian Mono inherits Martian Grotesk’s sharp personality while staying disciplined enough for product UI, code snippets, and dense interface text.

[ 02 ] variable

It ships as a variable mono family with width and weight range, so one type system can cover display, interface, and technical copy without style drift.

[ 03 ] dev_xp

Clear punctuation, firm counters, and stable rhythm keep the font useful in terminals, design systems, and multilingual product surfaces.
